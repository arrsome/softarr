Hash Verification

Enable optional external hash databases to improve trust ratings. Each lookup is best-effort -- failures do not block analysis.

VirusTotal

Checks SHA-256 hashes against 70+ antivirus engines. Flags releases as BLOCKED if any engine detects malware.

NIST NSRL

Checks hashes against the NIST National Software Reference Library. Boosts trust for known-good software. No API key required.

CIRCL hashlookup

Checks SHA-256 hashes against the CIRCL national hashlookup service. Boosts trust for known-good software. No API key required.

MalwareBazaar

Checks SHA-256 hashes against abuse.ch MalwareBazaar. Flags releases found in the malware database. No API key required.

MISP Warninglists

Checks hashes against MISP community warninglists of known-bad indicators. Flags suspicious releases. No API key required.
